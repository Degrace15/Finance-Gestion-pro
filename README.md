# 💰 Finances & Gestion Pro

## 📌 Description
**Finances Gestion Pro** est une application web de gestion financière développée avec PHP, HTML, CSS et JavaScript.  
Elle permet de suivre facilement les revenus, les dépenses et de gérer son budget en temps réel.

---

## ⚙️ Fonctionnalités
- ➕ Ajout de revenus
- ➖ Ajout de dépenses
- 📊 Calcul automatique du solde
- 📅 Historique des transactions
- ⚡ Mise à jour dynamique avec JavaScript
- 🧾 Interface simple et intuitive

---

## 🛠️ Technologies utilisées
- PHP (backend)
- MySQL (si base de données utilisée)
- HTML5
- CSS3
- JavaScript (frontend dynamique)
